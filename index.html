Bots

At Drawers we value freedom and openness - we are building on top of an open source protocol XMPP to support "client choice". In addition to Drawers, many other clients such as pidgin provides excellent communication. We extended it by launching "Bots API" to help third party clients to develop bots

Bots are simply Drawers account that can be public(searchable by everyone - Uber Cabs bot)/private(shell terminal). It can do anything such as remote command execution, booking a cab, preparing for an examination, dictionary, home power control (IOT). Build bot, make yours and others life easier, earn money. Checkout some sample bots in next section.

Command line execution

Ever tired of ssh everytime to your remote machine to check memory or grep logs. Drawers provide a simple bot to run your linux commands over chat. All you need is to run the cle bot on your machine and you have a monitor ready. It is bandwidth aware and works perfectly on 2g unlike ssh. Fork it here

Your Bot

We have open source libraries for all the major language. Checkout out our java, php, python, C++, C sdks. For all the other languages, any xmpp library with some wrapper code to match data types should be enough. These are simple echo bots and you can extend it by integrating rest apis, system commands, IOT processors. We loved and use latest technologies such as Java-8, Reactive programming (RXJava, RXPY), Docker, XMPP. Get your hands dirty.
